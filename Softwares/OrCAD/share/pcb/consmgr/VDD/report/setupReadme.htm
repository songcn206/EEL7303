Allegro Visual Design Differencing (VDD) report viewer setup

Before you begin using Allegro VDD report viewer some simple necessary installations and configurations need to be made.

Please follow the instructions below to configure your system:

  1. If you have an ADW installed

    1. Set Allegro environment variable to point to <ADW installation root>/tools/fet/projmgr/bin:

      Add the following line in the top of the env file in pcbenv directory

      set vdd_firefox_executable_path=<firefox path>

      where <firefox path> is a string like

      • "C:\Cadence\ADW1631\tools\fet\projmgr\bin" on Windows
      • or "/cadence/adw1631/tools/fet/projmgr/bin" on UNIX

  2. If you do NOT have ADW installation:
    1. Download and install Mozilla Firefox version 4.0
      2. (version 3.6 will aslo do)

    2. Set Allegro environment variable to point to firefox location:

      Add the following line in the top of the env file in pcbenv directory

      set vdd_firefox_executable_path=<firefox path>

      where <firefox path> is a string like

      • "C:\Program Files\Mozilla Firefox" on Windows
      • or "/usr/bin" on UNIX

After that is done, new VDD report viewer will be shown when VDD box is checked in import constriants dialog
